# Professional Portfolio | Amurut Mathew Koshy

A high-performance, responsive portfolio designed for Senior Product Management roles. This project utilizes a decoupled architecture where the UI is driven by a standalone JSON data structure.

## Deployment Link
The live portfolio is accessible at: [https://amurut.github.io/](https://amurut.github.io/)

## Technical Specifications
* **Frontend:** Tailwind CSS for modern utility-first styling and glassmorphic UI elements.
* **Logic:** Alpine.js for reactive data-binding and efficient DOM manipulation.
* **Data Layer:** Decoupled `data.json` source allowing for content updates without modifying core HTML.
* **Iconography:** FontAwesome 6.4 (SVG-based).

## Project Structure
* `index.html`: The core application file featuring a secure, dark-mode professional aesthetic.
* `data.json`: The single source of truth for all professional experience, academic history, and technical competencies.
* `README.md`: System documentation and deployment instructions.

## Key Features
* **Privacy-Centric Communication:** Direct contact information is omitted from the public UI; professional engagement is routed exclusively through a high-visibility LinkedIn integration.
* **Scalable Architecture:** Updates to professional milestones or skill sets are handled via JSON modifications, ensuring the code remains dry and maintainable.
* **Optimized UX:** Mobile-first responsive design tailored for executive review, featuring clear hierarchy and typography.
* **Domain Focus:** Structured to highlight leadership in Product Strategy, AI/ML integration, and Systems Engineering.

## Configuration and Updates
To update the portfolio content, modify the corresponding fields within `data.json`. The application will dynamically render the updated content upon the next session initialization.
